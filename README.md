## Projeto: Sistema de Vendas e Fidelidade — Cafeteria Geek "Byte & Brew"

O projeto consiste no desenvolvimento de um sistema de vendas e fidelidade para uma cafeteria com temática geek, chamada "Byte & Brew". O sistema oferece um gerenciamento completo dos itens e um sistema de cadastro de clientes, que ganham XP (pontos acumulativos) a cada compra realizada.
